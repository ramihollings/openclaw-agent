Initialize Hub

Access to the mr.technology Autonomous Hub is gated via Docker. Ensure your FOUNDER_KEY is set in your environment variables.

1. Pull the Core

docker pull mrtechnology/ecosystem-core:v4.2

2. Execute Boot

docker run -d --name mrtech-hub -e LICENSE_KEY=FOUNDER-XXX -p 8000:8000 mrtechnology/ecosystem-core
